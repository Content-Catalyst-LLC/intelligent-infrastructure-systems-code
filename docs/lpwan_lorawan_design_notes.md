# LPWAN and LoRaWAN Design Notes

LPWAN and LoRaWAN are useful for:

- remote assets
- battery-powered field nodes
- small telemetry payloads
- disaster monitoring
- environmental sensing
- rural infrastructure
- low-bandwidth status updates

Design considerations:

- payload size
- gateway coverage
- link budget
- duty cycle
- device class
- encryption
- provisioning
- battery life
- fallback communication
